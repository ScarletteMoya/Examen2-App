### **Descripción**  
Esta aplicación fue desarrollada como parte del Examen 2.  
Permite visualizar noticias del equipo de emergencias 911, mostrar información del estudiante y registrar eventos de emergencia.  

---

### **Funcionalidades**  
- Pantalla de inicio con presentación del equipo de emergencia y noticias.  
- Registro de eventos con título, descripción e imagen.  
- Lista de eventos guardados localmente.  

---

- React Native  
- Expo
